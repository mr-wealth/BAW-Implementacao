B.A.W website/app structure (MVP)
Goal: Allow sellers to register, list products and manage orders. Enable buyers to browse, purchase and track orders. Handle secure payments, global shipping and multilingual support.

Core features (MVP)
For Seller (SMEs)
* Sign up/log in
* Create store profile (logo/image, name, description, contact info, country, language)
* Product upload (name, description, price, photo, stock quantity)
* Basic inventory tracker
* Order mangement (view orders, mark as shipped, mark as delivered)
* Payment integration (e.g, paypal, cash options, other options)
* Mobile friendly dashboard
* Chat with buyers 
For Buyers
* Register/login
* Browse stores and products (search + categories)
* Product page (photos, description, price, “add to cart”, “buy now”)
* Cart + checkout
* Track orders
* Multicurrency display
* Contact seller button (via app chat)
Admin Dashboard
* View/manage orders
* Aprove/reject sellers accounts
* Track payment manually or via integration
